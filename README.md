# 勇者之塔 (Brave's Tower)

![License](https://img.shields.io/badge/License-MIT-green.svg)
![Genre](https://img.shields.io/badge/Genre-Roguelike-orange.svg)

一個充滿挑戰與未知的 Roguelike 爬塔冒險遊戲。

---

## 📖 遊戲簡介

在《勇者之塔》中，玩家將扮演一名孤傲的勇者，踏入不斷變幻的魔王高塔。你的目標只有一個：**抵達塔頂並擊敗魔王**。然而，這座高塔受到魔力影響，每一層的空間與事件都在不斷重組，唯有強大的實力與正確的決策才能讓你生存下去。

---

## 🎮 遊戲特色

### 🎲 純粹的 Roguelike 體驗
* **隨機生成地圖：** 每一層的房間配置、怪物分布與場景障礙皆由系統隨機生成，確保每次冒險都是全新的體驗。
* **動態難度系統：** 隨著樓層攀升，怪物的攻擊性與血量將持續成長，考驗玩家的應變上限。
* **You Only Live Once (YOLO)：** 嚴苛的死亡懲罰——一旦失敗，進度將全數歸零。每一次的生命都彌足珍貴。
* **特殊彩蛋：** 隱藏在其中的特殊彩蛋。

### ⚔️ 策略性的成長路徑
* **樓層獎勵抉擇：** 通關目前樓層後，玩家將面臨分歧點：
    * **抽取隨機事件：** 挑戰未知的下一層干擾（如敵人強化、環境陷阱），獲取高額回報。
    * **強化角色能力：** 穩健地提升基礎數值，為接下來的戰鬥打下基礎。
* **突發事件干預：** 每層樓都有可能觸發隨機事件（例如：強力的環境 Buff 或突如其來的強敵），打破重複性的戰鬥節奏。

### 👹 史詩級挑戰
* **Boss 戰：** 定期遭遇的高難度領主戰鬥，需要精準的操作與累積的屬性優勢方能克服。
